So what is the History of AI?

In 1955 Allen Newell and Herbert A Simon created the first artificial intelligence program called the Logical Theorist.
This program’s purpose was to deal with complex mathematical problems and prove theorems, this program successfully proved 38 of 54 theorems.
The logical theorist proved that a machine could work like a human and a base for AI future programming.

During 1956, the term “Artificial Intelligence” was created by John McCarthy at the Dartmouth conference, this was the first time that AI became a part of the academic field.

In 1966, researchers started emphasizing on developing algorithms that can solve mathematical problems, around this time the AI started to grow and take off as many researchers started looking into ways to use and implement AI.

During 1980 Expert systems were implemented into AI, this allowed the AI’s programming to simulate decision making abilities of a human expert.

In 2002 AI was first brought into the home with the Roomba vacuum.

During 2006 AI started to be implemented into the business world by companies like Facebook, Twitter and Netflix.

How does it work?

AI consists of three sections:

Machine Learning

“Machine learning is an application of artificial intelligence that provides systems the ability to automatically learn and improve from experience without being explicitly programmed”
(What is machine learning? A definition - Expert system. (2020, October 20))

Machine learning can access data and use it to learn, this starts with the observation of data through experiences and instructions.
Through this process, machine learning allows the computer to learn automatically to pick up on patterns and to make better future decisions.


Machine learning works on the below algorithms:
Supervised machine learning algorithms:
These algorithms can be used by taking the information from past instructions, being applied to new data from labelled examples to predict future events.
This algorithm starts with analysing a known dataset, from there the algorithm produces a function that helps predict output values.After sufficient training, the computer is then able to provide targets of any inserted data.
This algorithm can then compare its result to any correct results, so it can be compared and modify the system accordingly.

Semi-supervised machine learning algorithms
Semi-supervised machine learning algorithms is a mix between the supervised and unsupervised machine learning algorithms.
The purpose of this algorithm is to use both unlabelled and labelled data, this allows the system to considerably improve learning accuracy.This is only used when the data requires skilled and relevant resources to train it.

Unsupervised machine learning algorithms
This type of algorithm is used only when the example data is neither labelled or defined.
Having unsupervised machine learning algorithms shows how the system can infer a function over the unlabelled data.
This algorithm will not provide result data, but instead show how all the inputted data is related.


Reinforced Machine learning algorithms:
This algorithm interacts with its environment by producing actions and discovering errors and rewards.
This allows the system to determine the ideal behaviour in a specific context to maximise the performance.

(Fig 1.2a Machine learning example)

Deep Learning

“Deep learning is a subset of machine learning where artificial neural networks, algorithms inspired by the human brain, learn from large amounts of data”
((n.d.). Forbes.).

Deep learning uses a neural network to repeat a task multiple times, with each time the process being twerked a little to alter the outcome.
Deep learning is able to be used in the neural network of AI, as neural networks are built on multiple layers so the repeated tasks can be accomplished without holding up the rest of the network.
A popular example of deep learning in action is CNN or (convolution neural networks), the CNN involves learned features with 2D layers that helps this architecture suite to processing data.
This system works by extracting the features from the image itself, so every step of the process has a function to identify the image.

(Fig 1.3a Deep Learning example)

Neural Network

“A neural network (also called an artificial neural network) is an adaptive system that learns by using interconnected nodes or neurons in a layered structure that resembles a human brain.”
(What is a neural network? (n.d.))

The neural network combines multiple processing layers with simple elements operating in parallel. The network consists of an input layer, an output layer and multiple hidden layers.
This works with AI by allowing the layers to be used to transfer multiple signals across the system to process the algorithms. This system mimics how the nervous system works and is inspired by it.

    Expert Systems

    A large part of AI is the use and inclusion of an expert system.

    “Expert systems have specific knowledge to one problem domain” (Moné L, (2018, January 18)), this means that the expert system consists of knowledge that is related to people who use it ie. medicine, science, engineering.
    The knowledge is loaded from the expert system’s knowledge base, this contains all information regarding the subject. This information can be updated and removed along with new information can also be loaded into the system.
    AI uses expert systems as a reference of information, this allows it to pull up relevant information to solve problems.

    By having the ability to pull up relevant and usefull information, an AI system can help solve complex problems and sort through large amounts of data in quick time.
